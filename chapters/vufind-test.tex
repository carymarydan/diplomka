\label{sec:vufind-test}
Na tomto serveru probíhá veškerý vývoj systému VuFind a následné testování. Aktuálnì se zde upravuje a testuje nová verze VuFind 3.0.1.

Hardwarové vybavení tohoto stroje je:
\begin{verbatim}
pamìt RAM : 8 GB
úložný prostor pro databázi : 20 GB
úložný prostor pro data : 100 GB
úložný prostor pro systém : 20 GB
procesor : 4-jádrový 2,5 GHz Intel Core Processor (Haswell)
\end{verbatim}

Tento testovací server má jako operaèní systém nainstalován Red Hat Enterprise Linux Server release 7.1 (Maipo).
